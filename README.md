# DriveApp

Requirements
------------------------
•   npm

•    react-native

•    watchman




How to run   
--------------------------

```
  npm start
  
  react-native run-ios
  //requires xcode and ios simulator
  or
  
  react-native run-android
  
  // requires android environment setup and simulator
  
  ```
